# Active Directory Home Lab

## Overview
Built a fully functional Active Directory lab using VirtualBox to simulate a real enterprise IT environment.

## Tools Used
- Oracle VirtualBox
- Windows Server 2022 Evaluation
- Windows 11 (25H2)

## Lab Architecture
- **Domain Controller:** AD-Server (Windows Server 2022)
- **Client Machine:** AD-Client (Windows 11)
- **Domain Name:** lab.local
- **Network:** Internal Network (192.168.1.0/24)

## What I Built
- Installed and configured Windows Server 2022
- Promoted server to a Domain Controller
- Created Active Directory domain (lab.local)
- Created Organizational Unit (IT-Department)
- Created domain user account (jsmith)
- Installed Windows 11 and joined it to the domain
- Successfully authenticated using domain credentials

## Skills Demonstrated
- Virtual machine deployment
- Windows Server administration
- Active Directory configuration
- Domain user and OU management
- Network configuration
- Domain join process
