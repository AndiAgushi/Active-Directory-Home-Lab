# Lab 2: Help Desk Ticketing System (osTicket)

## Overview
Deployed a fully functional help desk ticketing system using XAMPP and osTicket to simulate a real IT service desk environment.

## Tools Used
- Windows Server 2022
- XAMPP (Apache, MySQL, PHP)
- osTicket v1.18.3
- phpMyAdmin

## What I Built
- Installed and configured XAMPP web server
- Created MySQL database for osTicket
- Installed and configured osTicket v1.18.3
- Created Help Topics (Hardware, Software, Network)
- Submitted and resolved help desk tickets
- Managed ticket workflow from open to resolved

## Skills Demonstrated
- Web server administration
- Database creation and management
- Help desk ticketing system configuration
- IT service desk workflow simulation
- Ticket triage and resolution
